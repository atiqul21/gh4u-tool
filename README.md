# gh4u-tool
আপনার ডিগ্রি, সাবজেক্ট ও CGPA অনুযায়ী বাংলাদেশে কি ধরনের চাকরি বা ক্যারিয়ার সম্ভব তা জানুন এই স্মার্ট ক্যারিয়ার অ্যানালাইজার টুলে। বাস্তবসম্মত ক্যারিয়ার সাজেশন, স্কিল উন্নয়নের পরামর্শ এবং মজার মন্তব্যসহ একটি আকর্ষণীয় ফটোকার্ড জেনারেট করুন যা সহজেই ফেসবুক ও অন্যান্য সোশ্যাল মিডিয়ায় শেয়ার করা যায়।
